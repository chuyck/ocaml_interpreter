CSE305 Programming Languages Interpreter Project. 

OCaml based interpreter for a simple stack-based language. 

Example Program:\n
push :unit:\n
push :error:\n
swap\n
push x\n
push num0\n
pop\n
push num1\n
push 8\n
bind\n
push num2\n
push 9\n
bind\n
push num3\n
push num4\n
toString\n
println\n
toString\n
println\n
toString\n
println\n
toString\n
println\n
toString\n
println\n
toString\n
println\n
toString\n
println\n
quit\n

num4\n
num3\n
:unit:\n
:unit:\n
